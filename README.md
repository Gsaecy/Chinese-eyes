# 🇨🇳 扩展选择助手 — 看透外文，一目了然

VS Code 扩展市场的智能翻译与选择助手。浏览 VS Code 扩展市场，自动将详情翻译为中文，AI 智能总结扩展功能、收费方式与使用指南。

> 🌟 **版本 0.3.4** — 翻译面板上线 + 多项体验优化

---

## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 🔍 **扩展搜索** | 浏览 VS Code 扩展市场，支持按关键词、热度、评分、更新时间排序 |
| 🌐 **简介翻译** | 自动将扩展标题和描述翻译为中文，支持多种翻译提供商 |
| 📖 **README 翻译** | 打开扩展详情时自动翻译完整 README 文档 |
| 🧠 **AI 智能总结** | 使用 DeepSeek / OpenAI 兼容 API 自动总结扩展功能、收费模式和使用指南 |
| 📊 **安装数据** | 显示扩展安装次数、评分和评价数量，辅助决策 |
| 🗂️ **翻译面板** | 独立翻译面板，粘贴原文即可翻译并生成摘要（适合翻译智能体插件等外部内容） |
| 🎯 **精确搜索** | 输入 `publisher.extension` 格式（如 `ms-python.python`）可精确按 ID 搜索 |

---

## 🚀 快速开始

### 1. 安装
在 VS Code 扩展市场搜索 **扩展选择助手** 或 **ext-trans-picker**。

### 2. 配置（可选）
- 默认使用**本地词典**（离线可用，无需 API Key）
- 如需 **AI 总结** 功能，推荐配置 DeepSeek：
  ```
  设置 → 选择 DeepSeek → 填入 API Key → 保存
  ```
- 也支持 OpenAI 兼容协议（阿里云 DashScope、Moonshot、Together 等）

### 3. 使用
- 点击活动栏 🧿 图标打开扩展列表面板
- 搜索扩展，点击「详情」查看翻译后的 README
- 点击「AI 总结」获取智能分析
- 点击标题栏 📝 图标打开翻译面板
